Website landing page Toko Online sederhana, dibuat dengan html, css, dan javascript. Dan json sebagai database sederhana nya.

Sistem kerja websitenya:
1. Calon pelanggan mengunjungi website.
2. Calon pelanggan tertarik dengan produk lalu akan diarahkan ke whatsapp, setelah mengklik tombol CTA.

Cara Clone ke Komputermu :

1 git clone "{url repositorinya}"

2. npm install

3. npm run dev
